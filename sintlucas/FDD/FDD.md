# FDD Avenoir - Clothing Catalogue

## General description
A clothing catalogue where people can add their own clothing pieces.

## Functional requirements

### Adding custom clothing pieces to the Website
- **Description:** Users should be able to upload pieces to the website

### Loading the garments. 
- **Description:** The page should dynamically show different clothing pieces.

### Dynamic page routing
- **Description:** Dynamic page routing so you can click on each garment to take you to another page.

### Public space. 
- **Description:** There should be a public place where users can find any (public) clothing pieces uploaded by other users. There should be a search by name and search by type fitler

### Filters. 
- **Description:** The user should be able to filter clothing by: Searching, Filtering type (shirts, pants, etc.) and further filtering

### Private board. 
- **Description:** Each user gets a private board which only they can see.

### Account Creating.
- **Description:** Users should be able to register and log into an account

## Non-functional requirements

### The page is responsive for desktop
- **Description:** The page is going to be responsive for desktop and maybe mobile devices later.

### The website should have a proper SEO
- **Description:** The website will have correct HTML tags for each element and improve loading speeds by only loading the neccessary elements. We are going to focus on not rerendering components when its unneccessary.
